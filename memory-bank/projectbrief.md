# Project Brief

Telegram Scheduler Mini App is a combined Telegram bot and Mini App for collaborative group scheduling.

Main product features:

- workspace setup through Telegram chats
- event creation and updates
- time voting through polls
- Google and Yandex calendar integration
- reminders and attendance tracking
- participation statistics

Repository goals:

- keep the codebase minimal
- preserve production functionality
- support direct server deployment without tunnel tooling
